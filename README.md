# PREPARE

## Vision
Real-time AI-assisted emergency triage platform that connects ambulance teams and hospitals to reduce response delay and surgical prep time.

## Tech Stack
- Next.js (App Router + TypeScript)
- TailwindCSS
- Supabase (DB + Auth)
- OpenAI
- OCR (Tesseract)
- PWA Support
